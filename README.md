# Parish Management System

A Flutter-based parish management application developed to simplify church and community administration.

## Features
- Family registration and member record management
- Secure login and authentication
- Church bulletin and announcements
- Community-related services
- User-friendly Flutter UI

## Tech Stack
- Flutter
- Dart
- SQLite
- SharedPreferences
- MVVM Structure
- Android / Web support

## Project Purpose
This project was developed to digitalize parish operations and improve the management of family and church member information.
